Brief Parameter Description:

Listed below are brief explanations of the parameters for UCLCHEM. For more details, we refer the user to the article for UCLCHEM and the Github Page for the source code of UCLCHEM.

Parameter Units Description Absolute Boundaries
Final Density cm-3 The density at which the model will stop collapsing 100 - 109 cm-3
Initial Temperature K The temperature the object has throughout the modeling. 5 - 30 K
Zeta factor 1.3e-17 s-1 Zeta is the factor by which the cosmic ray ionization rate is multiplied. See units for the standard CR ionization rate which will be multiplied by the zeta factor given by a user. 5 - 30 K
Radiation field Habing External radiation field affecting the object 1 - 104 Habing
Rout No Units 0.0001 - 10 pc
